Sobre Mi

Hola, Soy Yoselyn Bascuñan

Ingeniera en Informatica

Un desarrollador apasionado por construir soluciones tecnológicas que marcan la diferencia. He trabajado en diversos proyectos, incluyendo un CRM para una empresa de tapicería, donde pude aplicar mis conocimientos en desarrollo web y bases de datos. Realicé mi práctica profesional en AFP Capital, enfocándome en el modelamiento de datos, lo que me permitió desarrollar un pensamiento analítico y habilidades para manejar grandes volúmenes de información.

Tengo un sólido conocimiento en lenguajes como Python, Java, JavaScript, y React, además de experiencia trabajando con bases de datos relacionales. Me apasiona el trabajo en equipo y soy un firme creyente en las metodologías ágiles para la gestión de proyectos. Mi proyecto de título fue una página web de cotizaciones para artículos de computadora, donde pude combinar mis habilidades técnicas con un enfoque en la experiencia de usuario.

Siempre estoy buscando nuevos desafíos y oportunidades para seguir aprendiendo y creciendo como profesional. Mi objetivo es seguir desarrollando aplicaciones que ofrezcan soluciones innovadoras y de alto impacto.

Diseño de Paginas Web

HTML (HyperText Markup Language) es la base de cualquier sitio web, permitiendo estructurar el contenido con elementos semánticos claros. CSS (Cascading Style Sheets) se encarga del diseño y la presentación, aportando estilos personalizados para crear interfaces atractivas y responsivas. Bootstrap, un framework CSS, me permite acelerar el desarrollo mediante un sistema de grillas y componentes predefinidos, facilitando la creación de sitios web modernos y adaptables a diferentes dispositivos.

Base de datos

  • MySQL
  • PySQL
  • SQL Server
  • Xampp

SQL Server y MySQL son sistemas de gestión de bases de datos ampliamente utilizados para manejar datos estructurados de manera eficiente en aplicaciones de todo tipo. MariaDB es una versión mejorada y abierta de MySQL, ideal para proyectos de alto rendimiento. PySQL es útil para interactuar con bases de datos utilizando Python, permitiendo consultas dinámicas y automatización de tareas. XAMPP facilita la creación de entornos de desarrollo locales para pruebas y despliegue rápido de aplicaciones web.
